# 豆瓣电影
豆瓣电影客户端，目前只提供当前热映电影列表和相应详情页，采用MD。

## 架构相关
- 网联模块使用square全家桶，即[okhttp](http://square.github.io/okhttp/)+[retrofit](http://square.github.io/retrofit/)，使用gson序列化
- [RxJava](https://github.com/ReactiveX/RxJava) 的使用
